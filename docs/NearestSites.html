JTides Help | Nearest Sites



Click the "Find Nearest Sites" () toolbar button to see a list of sites that are near the currently displayed site.

The default list includes both tide () and current() sites. To choose only one of these categories, select your choice in the dropdown list at the upper right. The list will be recreated.

The list is sorted by distance, nearest first, with the currently displayed site at the top of the list. To choose one of the sites for display, simply click it with the mouse cursor.

To search for a site near a particular latitude and longitude, just enter that position into the latitude and longitude entry windows using this format: ddd.dd [mm.mmm] [n/s/e/w]
